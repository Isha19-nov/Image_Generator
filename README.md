# 🖼 AI Image Generator Website

A responsive web application that allows users to generate AI-based images using text prompts.

## 📌 Overview

The AI Image Generator is a web-based application built using HTML, CSS, and JavaScript.
It allows users to input text prompts and generate images dynamically through API integration.

This project demonstrates:

- API handling

- DOM manipulation

- Asynchronous JavaScript (fetch/async-await)

- Responsive UI design

## ✨ Features

- Text prompt-based image generation

- API integration

-  Loading animation

- Responsive design

- Download generated images

- Clean and modern UI

## 🛠 Tech Stack

- HTML5

- CSS3

- JavaScript 

- External Image Generation API

## 💻 System Requirements

- Modern Web Browser (Chrome, Edge, Firefox)

- Internet Connection (for API requests)
